# 📚 Repositório de Atividades da Faculdade

Este repositório foi criado para armazenar e organizar todos os exercícios, projetos e conteúdos práticos desenvolvidos durante as aulas da faculdade. O foco principal dos materiais aqui presentes é o desenvolvimento em **Java**, abrangendo desde os fundamentos da linguagem até conceitos avançados de Programação Orientada a Objetos (POO) e padrões de projeto.

## 🗂️ Estrutura do Repositório

Os materiais estão organizados por aulas e projetos, documentando a progressão do aprendizado:

* **`aula_1` a `aula_9`:** Fundamentos da linguagem Java, estruturas de controle, laços de repetição, manipulação de vetores/matrizes e os primeiros passos na criação de classes e objetos.
* **`aula_10` a `aula_18`:** Aprofundamento nos pilares da Programação Orientada a Objetos (Encapsulamento, Herança e Polimorfismo), uso de construtores e classes Wrappers.
* **`aula_19`:** Trabalhando com **Interfaces** e níveis de abstração no Java.
* **`aula_22`:** Manipulação de arquivos (Leitura e Escrita / I/O de dados).
* **`aula_23`:** Conceitos avançados incluindo **Serialização de Objetos** e introdução à arquitetura **MVC (Model-View-Controller)**.
* **`projeto1`:** Mini Projeto prático de um **Sistema de E-commerce** construído em Java, aplicando os conceitos de POO vistos no semestre.
* **`avaliacao1` e `revisao`:** Resolução de listas de exercícios, desafios práticos e preparação para provas.

## 💻 Tecnologias Utilizadas

* **Linguagem:** Java
* **Paradigmas:** Orientação a Objetos (POO)
* **Arquitetura:** MVC
